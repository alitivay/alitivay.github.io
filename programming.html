Here are some of my past commercial projects:

Booom.io (Year: 2016, ~242 Commits) [Request Demo]

Booom.io is an ambitious project that provides a multi-tenant platform for e-commerce. Users can sign up and receive their very own online store with just a few easy steps. The platform includes search services, intelligent product recommendation and secure trust-building mechanisms that ensure genuine transactions between buyers and sellers.

Technologies: Ruby on Rails, Redis, PostgreSQL, JQuery

Nnm-server (Year: 2016, ~328 Commits) [Request Demo]

Nnm-server is a real-time Websocket API that is used for fast storage, retrieval, and processing of data between a massive number of simultaneous users. A main application for this technology is realized in massively multiplayer online games (MMOs), where a large number of users need to play together and exchange data on various aspects of a game such as moves, turns, resources and in-game physics.

Technologies: Node.js, Redis, MySQL, Socket.io, Express.js

Irtos.org (Year: 2015, ~195 Commits) [Request Demo]

Diabetic Retinopathy is a serious damage to the retina, that can result in significant loss of vision in diabetic patients. These patients need to be screened regularly to prevent this complication. Often, the resources needed to do so exceeds the available specialists and equipment, especially in remote/under-developed locations. Irtos.org is an online platform aimed at fixing this problem. Patients in remote locations can go to local centers and get inspected. Their data is transferred securely to several graders to be inspected for diabetes-related complications. The system contains mechanism for automatic quality assurance, arbitration and conflict resolution.

Technologies: Ruby on Rails, PostgreSQL, Bootstrap, JQuery

Gardoone (Year: 2015, ~101 Commits) [Request Demo]

Gardoone is an online multi-tenant workflow managements system (WMS). In Gardoone, people can easily sign-up and instantly receive a web-based panel that enables them to create their own workflows. Gardoone's competitive advantage over similar products is that the interface and the calendar system is tailor-made for Persian users. A use-case example can be a manufacturing setting, where a set of design documents need to be automatically routed from designer to director and then to the production engineer. The WMS contains mechanisms for form-building, process-building, assignmet and delegation.

Technologies: Ruby on Rails, PostgreSQL, Bootstrap, JQuery
